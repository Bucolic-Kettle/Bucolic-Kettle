## Usage

> `npm start` in root directory to start web server. Ensure that backend server and postgresql server are already running. See [Bucolic-Kettle-Backend](https://github.com/Bucolic-Kettle/Bucolic-Kettle-Backend).

## Requirements

- Node
- Express
- Postgresql
- React/Redux
